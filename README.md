## Steps to launch
>  git clone https://github.com/simranxx9/MoviesPlaylist.git

>  cd MoviesPlaylist

>  cd backend<br />
    <b> npm install</b><br />
    <b> npm start</b>
 
> cd frontend<br />
    <b > npm install</b><br />
    <b> npm start</b>
